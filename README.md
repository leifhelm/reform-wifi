# WiFi Adapter for MNT Reform

This module is **untested**.

This adapter uses the PCIe and USB interfaces of the RCORE module, like the Pocket Reform.

The schematic is largely from [pocket-reform-wifi](https://source.mnt.re/reform/pocket-reform/-/tree/2fd691b93a8a8b05e923d17600f8ea777f0ab824/pocket-reform-wifi).
